# Customer Churn Prediction Using Machine Learning and Explainable AI

## Project Overview

This project develops a machine learning framework for predicting customer churn in digital banking environments. The study compares multiple machine learning models and incorporates Explainable AI (SHAP) to improve transparency and support customer retention decisions.

## Features

* Customer churn prediction
* SMOTE for class imbalance handling
* SHAP-based explainability
* Comparative analysis of machine learning models
* Hybrid ensemble framework
